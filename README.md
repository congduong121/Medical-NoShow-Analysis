🏥 Medical Appointment No-Show Analysis on Google Cloud (BigQuery)
An end-to-end data analytics project investigating over 110,000 medical records to identify key factors influencing patient no-show behavior and optimizing clinic operations.

🛠 Tech Stack
Google BigQuery (SQL): Processing large-scale datasets, performing feature engineering, and ensuring Data Anonymization (Hashing IDs) for privacy compliance.

Google Looker Studio: Designing interactive BI Dashboards for multi-dimensional data storytelling.

Google Cloud Sandbox: Scalable environment for cloud-based data warehousing.

📈 Key Insights
The "SMS Paradox": SMS reminders are highly effective for short-term appointments but lose significant impact when the waiting time exceeds 15 days.

Chronic Condition Trends: Patients with chronic diseases (Hypertension, Diabetes) exhibit a 15% higher compliance rate compared to the general population.

Lead Time Correlation: A direct positive correlation was found between the length of the waiting period and the likelihood of a no-show.

🔒 Data Privacy
Applied strict data ethics protocols by anonymizing sensitive patient identifiers to align with healthcare data protection standards.
